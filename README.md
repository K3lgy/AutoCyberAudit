# AutoCyberAudit
AutoCyberAudit est un toolkit Python d'audit de sécurité automobile permettant d'évaluer les systèmes connectés via OBD-II, CAN Bus, Bluetooth, Wi-Fi, firmware et API télématiques. Il génère des rapports PDF professionnels et s'appuie sur les bonnes pratiques ISO/SAE 21434, UN R155, AUTOSAR et TARA.
